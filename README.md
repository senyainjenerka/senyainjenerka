# 🦕

QA-инженер с опытом ручного и исследовательского тестирования мобильных (Android) и веб-приложений (FinTech / GameDev). Умею быстро погружаться в сложную логику проектов и находить неочевидные баги. Уверенно работаю с клиент-серверной архитектурой: глубоко тестирую API (Postman), использую Charles Proxy для работы с трафиком и пишу небольшие JS-скрипты для облегчения рутинных проверок. Хочу стать частью компании, в которой смогу развить свои профессиональные навыки, а также смогу повлиять на развитие классного и качественного продукта.
